# Eclipse to IntelliJ IDEA Transition Guide (Windows)

A concise mapping of the most common Eclipse shortcuts to their IntelliJ IDEA equivalents for developers making the switch.

## 🚀 Navigation & Search

| Action | Eclipse | IntelliJ (Windows) |
| :--- | :--- | :--- |
| **Search Everywhere** | `Ctrl + 3` | **`Double Shift`** |
| **Open Class / Type** | `Ctrl + Shift + T` | `Ctrl + N` |
| **Open File / Resource** | `Ctrl + Shift + R` | `Ctrl + Shift + N` |
| **Go to Declaration** | `F3` | `Ctrl + B` or `Ctrl + Click` |
| **Quick Outline** | `Ctrl + O` | `Ctrl + F12` |
| **Find Usages** | `Ctrl + Shift + G` | `Alt + F7` |

## 🛠️ Editing & Refactoring

| Action | Eclipse | IntelliJ (Windows) |
| :--- | :--- | :--- |
| **Quick Fix / Intentions** | `Ctrl + 1` | **`Alt + Enter`** |
| **Organize Imports** | `Ctrl + Shift + O` | `Ctrl + Alt + O` |
| **Format Code** | `Ctrl + Shift + F` | `Ctrl + Alt + L` |
| **Rename** | `Alt + Shift + R` | `Shift + F6` |
| **Delete Line** | `Ctrl + D` | `Ctrl + Y` |
| **Duplicate Line** | `Ctrl + Alt + Down` | `Ctrl + D` |
| **Move Line Up / Down** | `Alt + Up / Down` | `Ctrl + Shift + Up / Down` |
| **Generate Code** | `Alt + Shift + S` | `Alt + Insert` |

## 🐞 Running & Debugging

| Action | Eclipse | IntelliJ (Windows) |
| :--- | :--- | :--- |
| **Run Current File** | `Ctrl + F11` | `Shift + F10` |
| **Debug Current File** | `F11` | `Shift + F9` |
| **Step Over** | `F6` | `F8` |
| **Step Into** | `F5` | `F7` |
| **Resume** | `F8` | `F9` |

## 💡 Key Tips for Eclipse Veterans

1.  **Stop Hitting `Ctrl + S`**: IntelliJ saves automatically.
2.  **The "Eclipse" Keymap**: If muscle memory is too strong, go to `File` > `Settings` > `Keymap` and select **Eclipse** from the dropdown.
3.  **Find Action**: Can't find a setting or command? Press `Ctrl + Shift + A` and type the name of the action.
4.  **Project Structure**: In IntelliJ, a "Project" is equivalent to an Eclipse "Workspace," and a "Module" is equivalent to an Eclipse "Project."
